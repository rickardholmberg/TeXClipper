# Quick Start Guide

## Get Started in 3 Steps

### 1. Setup MathJax (Required)

Run the setup script to download MathJax:

```bash
./setup-mathjax.sh
```

### 2. Open and Build

```bash
open TeXClipper.xcodeproj
```

- Select your development team in Signing & Capabilities
- Press ⌘R to build and run

### 3. Grant Permissions

When prompted, grant Accessibility access:
- System Settings > Privacy & Security > Accessibility
- Enable TeXClipper

## Try It Out

1. Open any text editor (Notes, TextEdit, VSCode, etc.)
2. Type: `\int_0^\infty \frac{1}{1+x} dx`
3. Select the text
4. Press `⌘⌥K`
5. Watch it transform into rendered math!

Press `⌘⌥⇧K` to revert back to LaTeX.

## Keyboard Shortcuts

- **⌘⌥K** - Render LaTeX (display mode)
- **⌘⌥I** - Render LaTeX (inline mode)
- **⌘⌥⇧K** - Revert to LaTeX

## Menu Bar

Look for the ƒ (function) icon in your menu bar:
- Click for Settings window
- Test the renderer
- Quit the app
